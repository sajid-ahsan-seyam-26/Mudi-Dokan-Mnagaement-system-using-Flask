About This Project

Mudi Dokan Management System is a beginner-friendly Flask project. This website helps a small grocery shop manage products, stock, sales, profit and low-stock alerts.

Project Features:

  • View all grocery products
  • Search product by name and category
  • Add new product with buy price, sell price and stock
  • Sell product and automatically reduce stock
  • Restock products
  • View sales records and profit
  • Low stock alert
  • Delete product and sale records

What you will learn:

  • How Flask routes work
  • How HTML templates work with Jinja
  • How to use SQLAlchemy database
  • How to add, view, search, update and delete data
  • How to connect CSS and JavaScript with Flask
